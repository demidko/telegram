## Telegram-bot

Telegram bot template.

### Build

Execute `./gradlew clean shadowJar`. Your jar will be located at `./build/libs`.  
You can start bot with command `TOKEN=[your token] java -jar ./build/libs/cleaner-bot-all.jar`
